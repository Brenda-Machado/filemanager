# File Manager - Teste para Dev Junior Pixel Breeders

Aplicação web de gerenciador de arquivos:
    + Autenticação de usuários;
    + Upload/download/delete de arquivos;
    + Controle de acesso.

Desenvolvimento:
    + Frontend: React com TypeScript;
    + Backend: Python;
    + Banco de Dados: à escolha do candidato;
    + Armazenamento de arquivos: Local (filesystem).
